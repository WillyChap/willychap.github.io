Data Analysis

Loading Climate Data

Example: Working with NCEP/NCAR Reanalysis

import xarray as xr

# Open dataset
ds = xr.open_dataset('reanalysis.nc')

# Select region and time
data = ds.sel(
    lat=slice(25, 50),
    lon=slice(-125, -65),
    time='2023-01'
)

Statistical Analysis

Common operations for climate data:

  1. Time averaging: data.mean(dim='time')
  2. Spatial patterns: data.groupby('season').mean()
  3. Anomalies: data - data.mean(dim='time')
  4. Trends: Linear regression with scipy.stats

Visualization

Matplotlib Basics

import matplotlib.pyplot as plt
import numpy as np

# Generate sample data
time = np.arange(0, 24, 0.5)
temp = 15 + 5 * np.sin(2 * np.pi * time / 24)

# Create plot
plt.figure(figsize=(10, 4))
plt.plot(time, temp, 'o-', linewidth=2, markersize=4)
plt.xlabel('Hour of Day')
plt.ylabel('Temperature (¬∞C)')
plt.title('Diurnal Temperature Cycle')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

Mathematical Formulas

Atmospheric Equations

Ideal Gas Law:

\[ p = \rho R T \]

Hydrostatic Balance:

\[ \frac{\partial p}{\partial z} = -\rho g \]

Geostrophic Wind:

\[ u_g = -\frac{1}{\rho f}\frac{\partial p}{\partial y}, \quad v_g = \frac{1}{\rho f}\frac{\partial p}{\partial x} \]
